-- AppEngine with Eclipse
1. install PyDev in Eclipse: http://pydev.sourceforge.net/updates/
2. install python, config python interpreter in eclipse
3. import appengine project, set python-path in project properties:
   - google_appengine
   - google_appengine\lib\django
   - google_appengine\lib\webob
   - google_appengine\lib\yaml\lib
4. config run dialog
   - main module - d:\dev\google_appengine\dev_appsserver.py
   - args: ${project_loc} --port=9999
