Facebook API Change Log - v3.2

//
PageInfoData now extends EventDispatcher
- Added connectTimeout and loadTimeout to FacebookCall, so custom timeouts can be set per call.
- Added 2 events to DesktopSessionHelper && FacebookEvent
	- LOGIN_WINDOW_SHOW && PERMISSIONS_WINDOW_SHOW
- Added AIR Connect authentication code
- Added [Bindable] to FacebookArrayCollection.
- Added .name && .description properties to AttachmentData
- Made properties in MediaTypes static const's
- Fixed parsing of StreamMediaData.type && StreamMediaData.src
- Fixed AddTags.  x & y position will now be correctly uploaded. 
- Changed default image upload type to png
- FacebookError now extends FacebookData
- Change StoryType.VIDEO to correct id
- Removed VenueData (not needed), and replaced with FacebookLocation
- Moved parsing for FacebookLocation into xml parser utils.
- Deleted GenreData (Values match Documentation but inconsistent with API), changed to String instead. 
- Fixed some spelling mistakes in PageTypeValue
- Fixed PublishPost / HassAppPermissionValues
- Other minor bug fixes and tweaks

Added classes and packages:
com.facebook.utils.DesktopSessionHelper
